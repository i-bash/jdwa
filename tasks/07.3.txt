1. Написать функцию, которая выдаёт приветствие произвольному количеству людей:

welcome ('John', 'Peter', 'Mary')
> Welcome John, Peter, Mary!

Использовать arguments, заимствовать метод join объекта Array.


2. Создать класс person, имеющий свойство name (строка) и метод sayHello - функцию с одним параметром friends (массив). Метод sayHello для каждого элемента friend массива выводит строку <name> says hello to <friend>. Использовать метод Array.forEach (f), который поочерёдно вызывает f (element) для каждого элемента element массива.
