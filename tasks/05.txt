1. Создайте страницу, которая при нажатии кнопки "Ваше имя" будет запрашивать у пользователя его имя. При перезагрузке страницы имя должно отображаться на ней. При закрытии браузера имя забывается. Сделать через Cookies.

2. Модифицируйте страницу так, чтобы имя сохранялось после закрытия всех окон браузера и отображалось на странице при повторном запуске браузера. Добавьте кнопку "Забыть имя", стирающую информацию об имени.

3. Для формы из задания 03.1 при отправке запоминать значения переменных в виде объекта. Значения должны сохряняться при закрытии браузера и подставляться в форму при открытии страницы. Реализовать через Web storage.
